## 安装

```bash
npm i @fujia/ant

# or use yarn
yarn add @fujia/ant
```

## 使用

如何使用？从库@fujia/ant 中引入需要的算法直接使用即可。

```ts
import { quickSort } from '@fujia/ant';

const data = [3, 6, 1, 9, 16, 5, 2, 7, 4];

quickSort(data); // return:  [1, 2, 3, 4, 5, 6, 7, 9, 16]
```
