## Installation

```bash
npm i @fujia/ant

# or use yarn
yarn add @fujia/ant
```

## Usage

How to use? Simply to import useful functions from the library, as follows:

```ts
import { quickSort } from '@fujia/ant';

const data = [3, 6, 1, 9, 16, 5, 2, 7, 4];

quickSort(data); // return:  [1, 2, 3, 4, 5, 6, 7, 9, 16]
```
